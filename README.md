## How to use

### Obstacle tracker and extractor
```
ros2 launch obstacle_detector start_all_launch.py
```

### Obstacle tracker
```
ros2 launch obstacle_detector start_extractor_launch.py
```

### Obstacle extractor
```
ros2 launch obstacle_detector start_tracker_launch.py

```
